# SCC Scripts — Speedcubing Colombia CompScript

Scripts de [CompScript](https://github.com/cubingusa/compscript) para competencias organizadas por Speedcubing Colombia.

## Competencias

| Carpeta | Competencia | Fecha | Estado |
|---------|-------------|-------|--------|
| `2026-sac/` | WCA South American Championship 2026 | Jun 12-15, Bogotá | En desarrollo |

## Requisitos

- [CompScript](https://github.com/cubingusa/compscript) clonado al mismo nivel: `../compscript/`
- Node.js 18+
- [uv](https://docs.astral.sh/uv/) (para scripts de Python)

## Estructura

```
scc-scripts/
├── 2026-sac/         # SAC 2026 — ver 2026-sac/README.md para detalle
├── .gitignore
├── LICENSE
└── README.md
```

Cada carpeta de competencia es independiente. Ver el README dentro de cada una para instrucciones específicas.
